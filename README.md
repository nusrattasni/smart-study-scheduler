# Smart Study Scheduler

A **fullstack study scheduler app** built with **React, Node.js, Express, and MongoDB**.  
Organize your study tasks, track progress, and manage your schedule efficiently.

---

## Features

- **Add Tasks** – Create new study tasks with titles and optional descriptions.  
- **Delete Tasks** – Remove tasks you’ve completed or no longer need.  
- **Progress Tracking** – Keep count of completed tasks and see your overall progress.  
- **Persistent Storage** – Tasks are stored in **MongoDB**, so data stays even after refreshing the page.

---

## Tech Stack

- **Frontend:** React, JavaScript, CSS  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Others:** Axios for API calls
